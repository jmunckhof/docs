Every Mintlify site needs a `mint.json` file with the core configuration settings. Learn more about the [properties](#properties) below.

## Properties

<ResponseField name="activeModules" type="comma separated string">
  An array of active modules
  Example: `Products,FreeCount,Loss`
</ResponseField>

<ResponseField name="activeTabsProducts" type="comma separated string">
  In the Assortment module you can enable multiple tabs to be displayed when opening a [Product Detail View](https://docs.colleqtive.com/general/product-detail-view)
  
  Examples:
  <CodeGroup>
    ```json Single
    "activeTabsProducts": "Assortment"
    ```
    ```json Multiple
    "activeTabsProducts": "Assortment,StoreProductLogs,StoreLocations"
    ```
  </CodeGroup>
  <Expandable title="Options">
    <ResponseField name="Assortment" type="assortment">
    Usually this is the starting point of a product. Have a look at [Assortment Tab](https://docs.colleqtive.app/tabs/assortment)
    </ResponseField>
    <ResponseField name="Counting" type="counting">
    FreeCount tab
    </ResponseField>

  </Expandable>
</ResponseField>
